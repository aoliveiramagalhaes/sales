### Git basic commands

```shell
# initialize a repository (only for the first time)
git init

# add file
git add <FILE,>
# or ga

# verify status
git status
# or gst

# I know!
git commit -m "Message"
# gc -m

git log
# glg

# push changes to a remote repository (GitHub)
git push
# gp

# pull changes from GitHub
git pull
```
